# My Personal Portfolio 

 This is a personal portfolio designed to showcase my work, my skills, and a special gallery dedicated to my pets. 

I built this project to practice modern web design techniques and make it fully responsive, meaning it looks great on both computers and mobile phones.

## 🌟 Features

- **Interactive Project Section:** An accordion-style list where you can click to see details about my work.
- **Pet Gallery:** A beautiful grid layout showcasing my pets with hover effects.
- **Welcome Modal:** A friendly pop-up message that greets visitors when they enter the site.
- **Responsive Design:** The layout changes automatically from 1 column (mobile) to 3 columns (desktop) to fit your screen perfectly.
- **Dynamic Elements:** Buttons and cards that change color or size when you hover over them.

## 🛠️ Technologies Used

- **HTML5:** The "skeleton" or structure of the website.
- **CSS3:** The "style" and "makeup" of the site, including:
  - **Flexbox & CSS Grid:** For organizing the layout.
  - **CSS Variables:** To keep colors and fonts consistent.
  - **Media Queries:** To make the site mobile-friendly.
  - **Animations:** Smooth transitions and hover effects.
- **JavaScript:** The "brain" of the site that handles the pop-up window, the project accordion, and button clicks.

## 🚀 How to See the Project

For see the project please follow these steps:

1. Download the project files.
2. Find the file named `index.html`.
3. Right-click on it and select **"Open with"** -> Choose your favorite browser (Chrome, Edge, Firefox).
4. Enjoy the experience!
